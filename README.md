# Aplikasi Tiket Bioskop
Aplikasi ini digunakan untuk membuat sistem yang dapat membantu dalam manajemen tiket bioskop. 
Sistem ini memungkinkan pengguna memesan tiket film, memilih kursi, dan melihat riwayat transaksi. Dilengkapi dengan validasi input, formatting output, dan fitur log pemesanan.
